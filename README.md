# Как играть

1. Заказ товара через B
2. Товары приходит через 30 секунд
3. Их можно взять на E подойдя сзади к грузовику
4.После этого коробку можно поставить в стеллаж нажав E, надо подойти к стеллажу
5. Чтобы пришол покупател надо открыть магазин и хотябы в одном стеллаже должен быть товар. он приходит каждый 10 секунд
6. Когда покупатель пришол будет текст слева что он хочеть купить, вы должны взять этот товар подойдя к стеллажу и нажать E 
7. Бласт можно ставить только на огромный стеллаж
8. Все кроме бласта только к маленькому стеллажу
